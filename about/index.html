About

Info Builder

A personal information management app — define your own schemas, then create, browse, and explore your data across four views.

What it does

Create as many independent stores as you need (e.g. Contacts, Historical Events, Reading List). Each store gets its own entity types (e.g. Person, Organisation) and relationship types (e.g. WorksFor, Knows), each with custom field definitions.

Once your schema is set up you can create entities and relationships, then explore them through: a document view (the daily driver — inline editing, relationship display), a table view (sortable, searchable, paginated), an interactive graph (force-directed, filterable, focusable), and a timeline (auto-detected date fields, swimlanes, zoomable axis).

How it's built

Plain HTML, CSS, and vanilla JavaScript — no frontend framework, no build step. The API runs on Cloudflare Workers (TypeScript + Hono) backed by Cloudflare D1 (SQLite at the edge). The frontend is served from Cloudflare Pages with a Pages Function acting as an authenticated proxy to the Worker. Authentication is handled by Cloudflare Zero Trust.
